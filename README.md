# Newflix

Movie search app clone project, following React tutorials from Nomad Coders.

# Views

- Home
- TV Shows
- Search
- Detail

# API Verbs

- Now playing (Movie)
- Top Rated (TV)
- Popular (TV, Movie)
- Upcoming (Movie)
- Airing Today (TV)
- Search (TV, Movie)
- Details (TV, Movie) + Append to Response (related videos)
